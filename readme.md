# 벽돌깨기
  
벽돌 나오는 패턴이 여러개 정해져있고 그중에 하나를 불러와서 맵을 만듬  
전부 다 깨면 다음 맵으로 넘어감  
  
## 추가한 내용
1. 맵 추가
2. 파티클 추가
3. 폭발 화면 이펙트 추가
4. 타일추가 (폭탄에만 깨지는 특수 타일)
5. 화면 크기 자동 조정 (윈도우 크기를 바꿔도 최대한 자연스럽게 출력됨)
6. 점수 시스템 강화 (타일 파괴 점수 외에도 클리어시 남은 시간*2가 보너스 점수로 얻어짐, 빨리 클리어하면 더 높은 점수)

## 맵 형식  
oooo\nxxxx\noooo\nxxxx\noooo

- \n은 줄바꿈이고 풀어서 쓰면  
oooo  
xxxx  
oooo  
xxxx  
oooo  
  
# 현재 구현된 타일 종류  
- 'o' : 일반 타일, 닿으면 깨짐  
- 'x' : 공백  
- 'b' : 닿아도 안깨짐, 클리어 여부에 영향 x  
- '+' : 닿으면 공 3개 추가  
- '*' : 닿으면 공 3배로 복제
- 's' : 두번 닿아야 깨짐
- '>' : 속도가 빠른공 2개 추가
- 'i' : 폭탄 자기주변 3*3으로 블럭 파괴
- 'j' : 폭탄으로만 파괴됨
